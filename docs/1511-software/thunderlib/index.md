# ThunderLib

ThunderLib is a Java and C++ robot-side library for FRC robots that includes support for loading [ThunderAuto](../thunderauto/index.md) project files and running trajectories and autonomous modes.

ThunderLib provides the ability to read ThunderAuto project files and easily run trajectories and autonomous modes in only a few lines of C++ or Java code.
