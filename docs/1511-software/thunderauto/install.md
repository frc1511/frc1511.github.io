# Installing ThunderAuto

## Windows

ThunderAuto is available on the Microsoft Store:

[
![ThunderAuto Microsoft Store Page](https://get.microsoft.com/images/en-us%20dark.svg#only-light){width="200"}
![ThunderAuto Microsoft Store Page](https://get.microsoft.com/images/en-us%20light.svg#only-dark){width="200"}
](https://apps.microsoft.com/detail/9n2hbs3wm7cm?mode=direct)

Or you can download the `.exe` directly from the [Releases](https://github.com/frc1511/ThunderAuto/releases) page on GitHub.

## macOS + Linux

ThunderAuto also supports macOS and Linux, but must be built from source. Instructions for building ThunderAuto can be found in the GitHub Repository's [README](https://github.com/frc1511/ThunderAuto/blob/main/README.md).
