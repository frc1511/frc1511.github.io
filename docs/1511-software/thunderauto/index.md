# ThunderAuto

- [GitHub Repository](https://github.com/frc1511/ThunderAuto)

<figure markdown="span">
![Full App Screenshot](../../assets/thunderauto/full_app_1.png){width="600"}
</figure>


ThunderAuto is a tool developed by FRC Team 1511 to create trajectories for FRC Robots with holonomic (swerve) drivetrains to follow. ThunderAuto trajectories may either be exported to CSV files to be loaded onto the robot, or [ThunderLib](../thunderlib/index.md) may be used to read ThunderAuto project files directly on the robot.
